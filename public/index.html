Blob Proxy Interception Flow

            
            

              
1

              

                
Trigger Worker fetch()

                
The page initiates a standard network fetch('/shared-worker.js') to load the worker script.

              

            


            

              
2

              

                
Route Intercepted

                
Playwright intercepts the fetch request mid-flight via context.route().

              

            


            

              
3

              

                
Get Original Script

                
Playwright fetches the original unpatched script from the development web server.

              

            


            

              
4

              

                
Inject Monkey Patch

                
Playwright prepends the custom self.fetch routing patch and fulfills the request with the modified body.

              

            


            

              
5

              

                
Instantiate Trusted Blob

                
The page receives the patched script string, wraps it into a local Blob, and boots the worker via the blob: URL.

              

            


            

              
6

              

                
Bypass Chrome Sandbox

                
Since the blob URL shares the page's host, Chrome allows real API calls, while the patch mocks targeted endpoints internally.
